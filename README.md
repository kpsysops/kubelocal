# kubelocal

### HOW TO START
INSTALL PYTHON3: https://www.python.org/ftp/python/3.9.6/python-3.9.6-amd64.exe

DOWNLOAD ZIP: https://github.com/kpsysops/kubelocal/archive/refs/heads/master.zip

RUN AS ADMINISTRATOR: main.bat 

ENJOY!!!
